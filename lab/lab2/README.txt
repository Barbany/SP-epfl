MFSP Application Session: Sampling/Interpolation Operators in Positron Emission Tomography
==========================================================================================

To run the Jupyter notebook, please setup your environment as follows:

* Install Anaconda
* In a terminal, type the following:

  $ conda create --name mfsp_pet python=3.6.* matplotlib=3.0.* numpy=1.15.* scikit-image=0.14.* scipy=1.1.* tqdm=4.28.* jupyter=1.0.*
  $ (source) activate mfsp_pet
  $ jupyter notebook
